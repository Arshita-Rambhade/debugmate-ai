# DebugMate AI

DebugMate AI is a developer-focused web application that helps users understand programming errors more easily. Users can paste code snippets, compiler errors, or terminal outputs to receive simplified explanations, debugging guidance, and corrected code suggestions.

The goal of this project is to make debugging less confusing for beginners by presenting technical errors in a more understandable and structured format.

---

## Features

* Beginner-friendly error explanations
* Root cause analysis
* Suggested fixes and corrected code
* Syntax highlighted code blocks
* Screenshot upload support
* Copy-to-clipboard functionality
* Local history using localStorage
* Responsive modern UI

---

## Tech Stack

* React
* Vite
* Tailwind CSS
* JavaScript
* Gemini API

---

## Installation

### Prerequisites

* Node.js installed

### Setup

1. Clone the repository

```bash
git clone https://github.com/your-username/debugmate-ai.git
```

2. Navigate into the project folder

```bash
cd debugmate-ai
```

3. Install dependencies

```bash
npm install
```

4. Create a `.env.local` file and add your API key

```env
VITE_GEMINI_API_KEY=your_api_key
```

5. Start the development server

```bash
npm run dev
```
